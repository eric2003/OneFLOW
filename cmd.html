FOR

@echo off
for %%i in (1,2,3) do echo %%i
pause
results:
1
2
3

@echo off

for %%i in (a b c d) do (
  echo %%i
)
pause
results:
a
b
c
d

使用for /f命令从文件中读取每一行内容并进行处理：

@echo off

for /f "tokens=*" %%i in (myfile.txt) do (
  echo %%i
)
pause
results:
1
2 3 4
5 a

myfile.txt

1
2 3 4
5 a

使用for /f命令遍历文件夹中的所有文件：

@echo off

for /f "tokens=*" %%i in ('dir /b') do (
  echo %%i
)
pause
